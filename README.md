# Aashan & Nisha - 3rd Anniversary 💕

A Netflix-style interactive website celebrating 3 years of love, memories, and journey together.

## 🎬 Features

- **Netflix-inspired UI** - Familiar and beautiful interface
- **Profile Selection** - Personalized experience for Aashan and Nisha
- **Birthday Lock System** - Unlock with Nepali birthday dates
- **Video Memories** - Organized by years (Year 1, Year 2, Year 3)
- **Story Cards** - Text-based memories for special moments
- **Responsive Design** - Works on all devices

## 🚀 Tech Stack

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **Storage**: Cloudflare R2 (for videos and images)
- **Hosting**: Vercel
- **Assets**: 17 videos, 28 posters

## 📦 Deployment

All videos and images are hosted on Cloudflare R2 for fast, global delivery.
